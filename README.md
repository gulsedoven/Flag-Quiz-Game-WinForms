#  Flag Quiz Game (Geliştirilmeye Açık Sürüm)

Bu proje, orijinalinde üniversite dersim kapsamında bir **grup projesi** olarak geliştirilmiştir. 
Projenin temel iskeleti ve dosya okuma algoritması, proje ekibi tarafından oluşturulmuştur.

##  Benim Amacım ve Katkılarım (My Contributions)
Ben bu projeyi bir "vaka çalışması" (case study) olarak devraldım. Amacım, mevcut "Spagetti Kod" yapısını temizleyip, modern bir mimariye dönüştürmektir.

**Şu an üzerinde çalıştığım (bana ait) geliştirmeler:**
1.  **Refactoring:** Kodların okunabilirliğini artırmak ve yorum satırları eklemek.
2.  **Veritabanı:** Dosya sisteminden (File IO) SQL Veritabanına geçiş yapmak.
3.  **Yeni Özellikler:** Süre sayacı ve skor tablosu eklemek.
